# Blog Website with Authentication

A fully deployed full-stack blog application featuring secure user authentication, protected routes, and complete content management.

## 🚀 Live Demo
* **Frontend:*https://blog-website-five-omega.vercel.app/* 
* **Backend API:*https://blog-website-3-3ag4.onrender.com*

## ✨ Features
* **User Authentication:** Secure registration and login system.
* **JWT Security:** JSON Web Tokens used for secure user sessions.
* **Protected Routes:** Only authenticated users can access the dashboard and create content.
* **Full CRUD Operations:** Authenticated users can Create, Read, Update, and Delete blog posts.
* **Live Deployment:** Production-ready frontend and backend hosted online.

## 🛠️ Tech Stack
* **Frontend:*React* 
* **Backend:*Node.js* 
* **Database:*MongoDB* 
* **Authentication:*JWT (JSON Web Tokens)*
